# AI-Powered Content Generation System

This project is an AI-powered content generation system that takes raw product data as input and generates various types of content, such as FAQ pages, product pages, and comparison pages.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Channakeshava96/kasparro-ai-agentic-content-generation-system-Channakeshava-L.git
    ```
2.  Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  Place your raw product data in the `input/product_data.json` file.
2.  Run the main script:
    ```bash
    python main.py
    ```
3.  The generated content will be saved in the `output` directory.
